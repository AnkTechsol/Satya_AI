# Contributing to Satya_AI

Thank you for your interest in contributing to Satya — the AI Agent Progress Tracker & Truth Source Manager! Every contribution, no matter how small, helps make this project better.

## Code of Conduct

By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).

## How to Contribute

### 1. Fork the Repository

Click the **Fork** button at the top-right of the repository page.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/Satya_AI.git
cd Satya_AI
```

### 3. Set Up the Development Environment

```bash
pip install -r requirements.txt
pip install -e .
```

### 4. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 5. Make Your Changes

- Write clear, well-documented code
- Follow the existing code style
- Add tests for new functionality

### 6. Run Tests

```bash
pytest tests/
```

### 7. Commit Your Changes

```bash
git add .
git commit -m "feat: Add your feature description"
```

### 8. Push and Create a Pull Request

```bash
git push origin feature/your-feature-name
```

Then open a Pull Request on GitHub from your fork.

## Pull Request Guidelines

- Keep PRs focused on a single feature or bug fix
- Write a descriptive title and summary
- Reference any related issues
- Ensure all tests pass before submitting
- Be responsive to review feedback

## Good First Issues

Look for issues labeled [`good first issue`](https://github.com/AnkTechsol/Satya_AI/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22) to find beginner-friendly tasks.

## Questions?

Open a [Discussion](https://github.com/AnkTechsol/Satya_AI/discussions) or check the [README](README.md) for more information.

## Star the Project

If you find Satya useful, please consider giving us a ⭐ on GitHub! It helps others discover the project.
